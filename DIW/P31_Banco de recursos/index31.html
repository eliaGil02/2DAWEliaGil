Investigación de licencias

Una licencia indica las condiciones para usar un recurso, como si se puede descargar, compartir o modificar, entre otras opciones. Sirve para que los autores digan qué permiten hacer con su trabajo y para que los usuarios sepan cómo pueden usarlo sin problemas.

La diferencia principal entre Copyright y Creative Commons es que el Copyright no permite usar un recurso sin permiso del autor. En cambio, las licencias Creative Commons son más flexibles y tienen distintos niveles de libertad, permitiendo compartir, modificar o usar el contenido de forma más abierta según la licencia concreta.

Formatos de imagen para la web

En la web se usan varios formatos de imagen, y cada uno tiene sus ventajas. Esta tabla resume sus características para ayudar a elegir el formato adecuado según el tipo de imagen y el uso que se le quiera dar.

Formato Peso medio Transparencia Escalabilidad Uso recomendado
JPG Bajo No No Fotos del día a día
PNG Medio/alto Sí No Imágenes con fondo transparente
SVG Muy bajo Sí Sí Logos e iconos que deben verse bien en cualquier tamaño
WebP Muy bajo Sí No Imágenes pequeñas y optimizadas para cargar rápido
